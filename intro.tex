\section{Introduction}

Cette manipulation a pour but la caract\'erisation d'un module optique (OM), ainsi que la pr\'eparation du dispositif n\'ecessaire. 
\begin{center}
\fbox{
\begin{minipage}{0.75\textwidth}
Cela implique, dans un premier temps, de se familiariser avec le dispositif :
\begin{itemize}
\item \'etudier l'efficacit\'e des PMs,
\item calibrer l'ADC,
\item d\'evelopper la logique d'aquisition de donn\'ees.
\end{itemize}

Ensuite, en vous basant sur les donn\'ees, vous devrez calculer :
\begin{itemize}
\item le gain $G$ de l'OM,
\item la r\'esolution $\sigma_\mathrm{G}$ de l'OM,
\item le nombre moyen de photo-\'electrons $\langle n_{\mathrm{pe}}\rangle$ produit par trigger dans l'OM.
\end{itemize}
\end{minipage}
}
